# Essence Verte - Perfumaria Premium

Site profissional e elegante para a Essence Verte, uma perfumaria que oferece fragrâncias premium de luxo.

## Características

- **Design Moderno e Minimalista**: Interface elegante com paleta verde, dourado e branco
- **Responsivo**: Totalmente otimizado para dispositivos móveis e desktop
- **Quiz Interativo**: Ferramenta para descobrir o perfume ideal
- **Catálogo Dinâmico**: 7 perfumes com descrições detalhadas
- **Integração WhatsApp**: Compra direta via WhatsApp
- **Redes Sociais**: Links para Instagram e WhatsApp flutuantes

## Perfumes Disponíveis

1. **J'ADORE** - Feminino (Dior)
2. **FAHRENHEIT** - Masculino (Dior)
3. **YARA** - Feminino (Lataffa)
4. **SAUVAGE** - Masculino (Dior)
5. **MONT BLANC FEMME** - Feminino (Mont Blanc)
6. **HACIVAT** - Unissex (Nishane)
7. **212 MEN** - Masculino (Carolina Herrera)

## Tecnologias

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Animações**: Framer Motion
- **UI Components**: shadcn/ui
- **Ícones**: Lucide React
- **Tipografia**: Google Fonts (Playfair Display, Lato)

## Deploy

O projeto pode ser facilmente deployado na Vercel:

\`\`\`bash
npm run build
\`\`\`

Depois faça push para o GitHub e conecte à Vercel para deploy automático.
